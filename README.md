# 🔐 Uteast MD5 Hasher (md5hasher)

A simple, fast, and lightweight CLI tool to convert passwords into MD5 hashes using Python.

---

## ⚡ Features

- 🔒 Instant MD5 hash generation  
- 🖥️ Works on Termux & Linux  
- ⚙️ Lightweight (no extra libraries)  
- 💻 Clean CLI interface  
- 🚀 Easy one-line installation  

---

## 📦 Installation (One-Line Setup)

```bash
git clone https://github.com/UTEAST/Md5hasherut.git && chmod +x Md5hasherut/md5hashgen.py && mv Md5hasherut/md5hashgen.py $PREFIX/bin/md5hasher && rm -rf Md5hasherut
```

---

##▶️ Usage
```bash
md5hasher
```
